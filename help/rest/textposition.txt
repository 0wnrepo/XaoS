.. _textposition:

Text position
=============

**Syntax**:(textposition keyword keyword)

Select `text`_ position. The first keyword specifies the horizontal position,
the second the vertical position. The horizontal position should be one of
'left, 'center, and 'right. The vertical should be one of 'top, 'middle, and
'bottom.

**Available as**: command line option, command
